Detailed Description

a Field instance represents e.g.

a field in a structure, a variable/register in a model or anything that can be represented as a sequence of up to 64 bits

◆ AccessMode

enum etiss::VirtualStruct::Field::AccessMode

defines how a field should perform reads/writes.

default: PREFER_LAMBDA

Enumerator
VIRTUAL 

uses the virtual _read() / _write functions

LAMBDA 

uses the lread,lwrite functions

PREFER_LAMBDA 

tries to use the lread,lwrite functions and falls back to virtual functions
